# 文本任务与翻译

## 目的
管理主文本、认领任务、翻译编辑与锁定。

## 模块概述
- **职责:** 主文本列表、筛选分页、认领、锁定、翻译保存、变更历史
- **状态:** 📝规划中
- **最后更新:** 2026-01-30

## 规范

### 需求: 主文本维护与认领
**模块:** 文本任务与翻译
支持按 fid + part 认领任务，支持多人认领但单人编辑锁定。

#### 场景: 进入翻译页锁定
- 用户从列表进入翻译页
- 系统锁定该文本，避免多人同时编辑

## API接口
### [GET] /texts
**描述:** 列表与筛选
**输入:** fid/part/人员/状态/关键字/分页
**输出:** 主文本列表

### [POST] /locks
**描述:** 锁定文本
**输入:** text_id
**输出:** 锁定结果

## 数据模型
### text_main
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| fid | varchar | 文件标识 |
| part | varchar | 分段标识 |
| source_text | text | 原文 |
| translated_text | text | 译文 |
| status | varchar | 状态 |
| edit_count | int | 变更次数 |

### text_changes
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| text_id | bigint | 关联主文本 |
| before_text | text | 变更前 |
| after_text | text | 变更后 |
