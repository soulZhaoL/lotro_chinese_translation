# 用户与权限

## 目的
提供登录、角色与权限管理能力。

## 模块概述
- **职责:** 用户登录、权限分配、角色管理、游客访问
- **状态:** 🧪实现中
- **最后更新:** 2026-01-30

## 规范

### 需求: 用户管理
**模块:** 用户与权限
提供用户登录、角色分配、权限可配置的基础能力。

#### 场景: 账号登录
- 用户输入用户名与密码
- 系统按配置的 hash_algorithm 校验并返回用户角色/权限

## API接口
### [POST] /auth/login
**描述:** 用户登录
**输入:** username, password
**输出:** 用户信息与角色列表

## 数据模型
### users
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| username | varchar | 用户名 |
| password_hash | varchar | 密码哈希 |
| password_salt | varchar | 盐 |
| is_guest | boolean | 是否游客 |

### roles
| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键 |
| name | varchar | 角色名 |
