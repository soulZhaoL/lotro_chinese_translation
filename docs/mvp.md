# MVP 联通步骤

## 目标
完成最小可验证链路：登录 → 主文本列表 → 翻译页锁定 → 词典查询

## 步骤 1：准备后端
1. 配置 `.env`
2. 执行迁移脚本
3. 启动后端服务（8000）
4. 打开 `http://localhost:8000/docs` 确认 API 可访问

## 步骤 2：准备前端
1. 配置 `web/.env`（关闭 Mock）
2. 启动前端开发服务（5173）

## 步骤 3：准备测试数据（最低集）
在数据库中插入：
- 一个用户 + 角色/权限
- 至少一条 `text_main`
- 可选：词典数据

## 步骤 4：联通验证
1. 前端登录
2. 列表页能拉到主文本
3. 点击进入翻译页，提示锁定成功
4. 词典页能获取并新增词条

## 说明
- 若需要前端独立验证，先开启 `VITE_USE_MOCK=true`
- 联通阶段必须使用真实后端
